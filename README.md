# Mukul Sherekar — Personal Website

Professional portfolio at the intersection of protein biochemistry and AI.

## Project Structure

```
├── index.html                 ← Main page (edit sections here)
├── assets/
│   ├── css/style.css          ← All styling (colors, layout, responsive)
│   ├── js/main.js             ← Interactivity (menu, scroll effects)
│   ├── images/                ← Profile photo and any future images
│   └── documents/             ← Resume PDF
└── vercel.json                ← Vercel deployment config
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **Add New Project**
3. Import your GitHub repo (`msherekar/msherekar.github.io`)
4. Framework Preset: **Other** (it's plain HTML)
5. Click **Deploy** — done!

### Custom Domain (optional)

In Vercel dashboard → **Settings → Domains** → add `mukulsherekar.com` (or any domain you own).

## Local Development

Just open `index.html` in your browser. No build step, no dependencies.

```bash
open index.html
```
